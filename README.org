* dot-emacs (.emacs.d)

Personal Emacs configuration for that1guycolin. Uploading it publicly in case someone else finds it
useful. Feel free to modify this config as much as you'd like for your own use, and definitely share
any modifications you make (I'm always looking for ways to improve)!
